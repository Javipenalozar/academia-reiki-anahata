# Academia de Reiki Anahata

Landing page estatica para el nuevo grupo presencial de Reiki Anahata que inicia el 22 de julio.

URL publica: `https://anahata.eventosjv.com/`

## Publicacion en Netlify

1. Sube este proyecto a GitHub.
2. En Netlify, crea un nuevo sitio desde ese repositorio.
3. Usa esta configuracion:
   - Build command: dejar vacio
   - Publish directory: `.`
4. Conecta el dominio `eventosjv.com` desde la configuracion de dominios de Netlify.
